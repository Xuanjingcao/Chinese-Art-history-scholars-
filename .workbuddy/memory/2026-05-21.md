3. **P1: auth.ts 老用户数据迁移隐患** — 旧格式 `{username}` 迁移时不再把 username 当 userId，而是直接清除让用户重新认证
4. **P1+P2: 清理冗余** — 删除 `main.tsx` 无用的 BrowserRouter；删除 `ProfessorModal.tsx` 重复的 AuthUser 类型定义，改为从 `@/lib/auth` 导入
